# Proyecto Web (Scaffold)

Pequeño proyecto estático de ejemplo.

Cómo probar:

- Abrir `index.html` directamente en el navegador.
- O ejecutar el servidor Node incluido:
- Abrir `index.html` directamente en el navegador (rápido, sin dependencias).

Opciones de desarrollo (recomendado):

1) Usar Vite (dev server con hot-reload)

```bash
npm install
npm run dev
```

2) Build y preview (para probar producción con Vite):

```bash
npm run build
npm run preview
```

3) Ejecutar servidor Node simple (producción básica):

```bash
npm install
npm start
```

Archivos creados:
- `index.html`, `styles.css`, `script.js`, `server.js`, `package.json`, `.gitignore`.
